# 태승전자(주) TSELEC - 기업 홈페이지

마이크로컨트롤러 설계·제조 전문 기업 태승전자(주)의 공식 홈페이지입니다.

## 구조

```
index.html          # 메인 홈페이지
admin.html          # 관리자 대시보드 (CMS)
pages/              # 서브페이지 (11개)
  ├── greeting.html         # 인사말
  ├── history.html          # 회사연혁
  ├── facility.html         # 회사전경
  ├── certifications.html   # 인증현황
  ├── organization.html     # 조직도
  ├── directions.html       # 오시는 길
  ├── products.html         # 제품영역
  ├── main-controller.html  # 메인 컨트롤러
  ├── display.html          # 디스플레이
  ├── others.html           # 기타 솔루션
  └── process.html          # 공정도
assets/images/      # 이미지 리소스 (44개)
```

## 기술 스택

- 순수 HTML/CSS/JavaScript (빌드 도구 없음)
- Google Fonts: Orbitron, Noto Sans KR
- Canvas 기반 회로 네트워크 애니메이션
- IntersectionObserver 스크롤 애니메이션
- 반응형 디자인 (1200px / 900px / 600px breakpoints)

## 배포

Vercel 정적 호스팅으로 배포됩니다.
